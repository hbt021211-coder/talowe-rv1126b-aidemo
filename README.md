🛠️ Talowe-Rv1126b-AI-Application-Demo  
推理引擎：RKNN(Rockchip Neural Network)  
硬件平台：RV1126B 芯片  
图像处理：OpenCV  
深度学习：ONNX/Pytorch/Caffe/Darknet/Caffe  

1. 人体姿态检测 (talowe_yolov8_pose_cam)  
功能：实时检测人体关键点，并进行姿态识别与行为分析  
应用场景：工业作业行为分析、运动姿态评估、跌倒检测与预警  

2. 通用目标检测 (talowe_yolov5_detection_cam)  
功能：通用目标检测，支持COCO数据集80个类别  
应用场景：智能监控、物体识别、自动驾驶  

3. 通用目标检测 (talowe_yolov26_detection_cam)  
功能：通用目标检测，支持COCO数据集80个类别  
应用场景：智能监控、物体识别、自动驾驶  

4. 通用目标检测 (talowe_yolov11_detection_cam)  
功能：通用目标检测，支持COCO数据集80个类别  
应用场景：智能监控、物体识别、自动驾驶  

5. 人脸五官检测 (talowe_retinaface_cam)  
功能：实时检测人脸关键点，实现五官定位与人脸特征  
应用场景：人脸识别预处理、身份验证、表情分析、智能门禁与考勤系统  

6. 文字检测识别 (talowe_ppocr_cam)  
功能：实时检测图像或视频中的文字区域，并进行字符识别与内容提取  
应用场景：车牌识别、票据与文档数字化、工业标签读取、智能巡检与信息录入  

7. 安全帽检测 (talowe_helmet_detection_cam)  
功能：实时检测作业人员是否佩戴安全帽识别  
应用场景：工地安全监控、工厂作业区域监管、高危作业场景安全预警  
 
8. 垃圾分类检测 (talowe_garbage_classification_cam)  
功能：实时识别垃圾类型（如可回收物、有害垃圾、厨余垃圾、其他垃圾），并进行分类标注  
应用场景：社区垃圾分类投放点监管、公共场所垃圾分类引导、校园垃圾管理系统  

9. 火焰烟雾检测 (talowe_fire_detection_cam)  
功能：实时检测画面中的火焰与烟雾目标，并进行异常告警与区域标注  
应用场景：森林防火监测、工厂仓库消防预警、园区安全监控、机房与高危区域火灾预防系统
